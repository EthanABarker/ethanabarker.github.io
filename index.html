About Me!

About Me

Hello there! My name is Ethan Barker and I am currently studying BSc(Hons)Computer Games Programming
at Staffordshire University. I started programming at the age of 10 beginning with small text based games in Python. On this website you can find some of the projects I've worked on for both personal and academic purposes.

Games Engine Creation

In Games Engine Creation, I was tasked with creating a platformer like Mario in C++ using the SDL library. In this module I practiced my C++ skills with tasks like handling collisions, loading maps and making multiple playable characters using polymorphism.

In Another Castle

While studying at College I was tasked with creating a platformer game in Unreal Engine 4. The result was the In Another Castle game I created. This cyberpunk inspired platformer demo was one of my first forte into level design as well as the UE4 Blueprint System.

F.O.G.S

In my first semester at university I made a short project using C++ along with Staffordshire University's in house library S2D. In this project I made a playable character who could cycle through a large animation sheet depending on player input.
